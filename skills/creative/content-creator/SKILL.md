# Content Creator Skill (内容创作者助手)

## 概述
这是一个专为**短视频/自媒体内容创作者**设计的Agent技能模板。开箱即用，将kunming-agent配置为一个完整的AI内容创作工作台。

## 核心能力

### 🎬 完整视频制作流水线
```
选题 → 文案生成 → 语音配音 → 封面图 → 字幕 → 视频合成 → BGM混音
```

每一步都有对应的免费工具，无需任何付费API。

### 📊 5种爆款内容模板
| 模板 | 适用场景 | 结构 |
|------|---------|------|
| **故事叙述型** | 历史解说/人物传记/案件分析 | 钩子→冲突→展开→高潮→金句 |
| **清单盘点型** | 知识科普/产品推荐/技巧合集 | 数字标题→递进要点→总结CTA |
| **热点评论型** | 新闻点评/社会现象/娱乐八卦 | 事件描述→观点抛出→多角度分析 |
| **干货教程型** | 技能教学/软件教程/生活妙招 | 痛点→效果预展示→步骤拆解→避坑 |
| **情感共鸣型** | 情感故事/人生感悟/正能量 | 场景代入→情感递进→共情爆发 |

## 工具使用指南

### 第一步：获取创作计划
调用 `content_pipeline` 工具：
```json
{"action": "generate_plan", "topic": "你的选题", "style": "storytelling", "platform": "douyin"}
```
返回完整6步工作流计划。

### 第二步：生成文案（LLM内置）
直接让AI根据模板结构撰写文案。参考各平台的文案风格要求。

### 第三步：语音配音
调用 `text_to_speech` 工具：
- 引擎：Edge TTS（免费）
- 推荐中文女声：`zh-CN-XiaoxiaoNeural`
- 推荐中文男声：`zh-CN-YunxiNeural`
- 输出格式：MP3（自动转Opus用于Telegram）

### 第四步：生成封面图
调用 `cover_generate` 工具：
- 风格选项：`modern`(默认), `gradient`, `bold`, `simple`
- 尺寸：1080×1920（竖屏）或自定义
- 配色方案已针对短视频平台优化

### 第五步：生成字幕
调用 `srt_generate` 工具：
- 将文案按段落分割为segments
- 自动根据音频时长计算时间轴
- 支持中英文混排，自动识别字符类型调整速度

### 第六步：合成视频
调用 `video_assemble` 工具：
- 输入：图片列表 + 音频文件 + SRT文件
- 输出：MP4竖屏/横屏视频
- 可选：烧录字幕到视频中

### 第七步：（可选）添加BGM
调用 `audio_mix` 工具：
- 主音频：配音文件
- 背景音乐：本地音乐文件
- BGM音量自动设为15%，不压过人声

## 经验学习集成

每次完成一个内容创作任务后，系统会自动记录：
- 什么类型的选题用什么模板效果最好
- 哪种TTS声音适合哪种平台
- 用户对哪些结果给出了正面反馈

下次做类似内容时，系统会优先推荐历史成功的方案。

## 使用示例

### 示例1：快速制作一条抖音短视频
```
用户: 帮我做一个关于"AI如何改变普通人生活"的抖音视频

Agent执行流程:
1. content_pipeline(generate_plan, topic="AI改变普通人生活", platform="douyin")
   → 返回6步计划和故事型模板
2. [LLM] 根据5段式结构生成文案（约300字）
3. text_to_speech(text=文案, voice="zh-CN-XiaoxiaoNeural")
   → 生成MP3语音文件
4. cover_generate(title="AI时代普通人如何逆袭", subtitle="第3个方法你绝对想不到", style="modern")
   → 生成1080×1920封面图
5. srt_generate(segments=[文案段落列表], audio_path=语音文件)
   → 生成带正确时间轴的SRT
6. video_assemble(images=[封面图], audio=语音, subtitles=SRT文件, resolution="1080x1920")
   → 生成最终MP4视频
7. audio_mix(primary_audio=语音, background_audio="bgm.mp3")
   → （可选）添加背景音乐
```

### 示例2：批量生产小红书图文
```
用户: 帮我做3条关于"居家收纳"的小红书内容

Agent:
1. content_pipeline(generate_plan, platform="xiaohongshu", style="listicle)
2. 对每个主题重复: 文案→封面图(横版)→排版输出
3. experience_record() 记录哪些主题效果好
```

## 最佳实践

### 文案写作原则
1. **前3秒定生死** — 开头必须有冲突、悬念或强烈情绪
2. **每句话都要有信息量** — 废话=划走
3. **口语化但不低俗** — 像跟朋友聊天一样自然
4. **控制节奏** — 短句+长句交替，避免单调

### 封面图设计原则
1. **对比色醒目** — 深底亮字或亮底深字
2. **文字少而精** — 主标题≤12字，副标题≤20字
3. **留白要够** — 不要填满整个画面
4. **品牌一致性** — 固定配色和字体建立辨识度

### 平台差异化策略
| 抖音 | 追求完播率，开头必须炸裂 | 语速偏快，信息密度高 |
| 小红书 | 追求收藏和分享 | 详细、有用、有美感 |
| B站 | 追求弹幕互动 | 可以有槽点、有梗、有争议点 |
